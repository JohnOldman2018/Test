Правила CSS

Текущей версией является CSS3. Несмотря на то, что спецификация CSS4 уже разработана, она пока поддерживается не всеми основными браузерами, поэтому текущей версией признается CSS3.

Универсальный селектор и селекторы типов

<style>
  * {text-align: center;} //универсальный селектор (весь текст будет
 выровнен по центру, элементы, не содержащие текст, не выравниваются)
  hr {width: 50%;} //селектор элемента горизонтальная линия
  h2, p {font-style: italic; color: blue;} //комбинированный селектор
</style>

Селектор класса

<style>
  .red {background-color: tomato;}
  .white {background-color: white;}
  .blue {background-color: skyblue;}
  q {font-family: Impact;}
</style>

Селектор класса с приставкой из типа элемента

  q.blue {background-color: skyblue;}
Применяется к классу blue только в элементах цитирования <q></q>

Каскадные таблицы стилей

Для того, чтобы не возникло противоречий при примененеии правил CSS, в данном своде правил существует приоритетность применения правил.
Следующий список показывает приоритет различных уровней правил стиля, начиная с самого высокого приоритета к самому низкому:

  1. Атрибут style в любом элементе
  2. Элемент style в разделе веб-страницы head
  3. Правила во внешнем файле
  4. В настройках пользователя для установленного браузера
  5. Во внутренних настройках браузера по умолчанию
Пример применения атрибута style только в одном из элементов <h2></h2>:
<h2 style="text-decoration:underline;">Welcome!</h2>
Результат отображается в браузере следующим образом:

Welcome!

Но такого применения следует избегать, хотя его часто можно встретить в устаревшем коде (legacy code). Такое применение сильно затрудняет дальнейшую поддержку веб-страницы, поскольку необходимо производить изменения во всех отдельных элементах веб-страниц, а не в одном месте в элементе стиля или в файле стиля. Раньше такое применение называлось встроенными стилями (inline styles).
Тем не менее, правило, установленное в виде атрибута style, будет иметь более высокий приоритет, чем правило, установленное в контейнере style в соответствии с приоритетами в каскадной таблице стилей.

Для указания на внешний файл стилей в отдел <head></head> веб-страницы вставляют элемент <link></link> с атрибутами rel и href:
<link rel="stylesheet" href="name-of-external-file">
Атрибут rel указывает браузеру на тип файла стилей, а атрибут href указывает браузеру на местоположение и название файла стилей.

Чем выгодно использование внешнего файла стилей?
Оно позволяет использовать общие правила CSS для всех веб-страниц проекта, что делает их единообразными.

Организация W3C больше не использует название "внешний файл стилей" (external style sheet), но оно прочно устоялось среди веб-разработчиков.

Цветовые свойства

Цвет в свойстве стилей color указывается разными способами:
  • .black {background-color: rgb(0%,0%,0%);} - с указанием знака процента насыщенности трех цветов
  • .favorite1 {color: rgb(127,127,127);} - с указанием степени цветовой насыщенности в диапазоне от 0 до 255
  • .favorite2 {color: rgba(127,127,127,0.5);} - с указанием степени насыщенности цвета в диапазоне от 0 до 255 и степени непрозрачности (opacity) в диапазоне от 0 до 1
  • .sapphire {background-color: #0f42ba;} - с указанием степени цветовой насыщенности в диапазоне от 0 до f (шестнадцатиричная система счисления)
  • p {background-color: hsl(120,50%,75%);} - с указанием цвета (hue) в диапазоне от 0 до 360 градусов по цветовому колесу (red - 0/360, yellow - 60, green - 120, cyan - 180, blue - 240, magenta - 300); степени насыщенности цвета (saturation) в диапазоне от 0 до 100%; степени яркости цвета в диапазоне от 0 до 100%
  • .background {background-color: hsla(120,50%,75%,.5);} - та же система hsl, но с добавлением степени непрозрачности (opacity) в диапазоне от 0 до 1

Свойства шрифта

Свойство font-style

Свойство font-style имеет следующие значения:

  • normal - вертикальный
  • oblique - наклонный
  • italic - курсив
  • initial - изначальный
Font-style: normal

Свойство font-variant

Свойство font-variant определяет вид строчных букв на экране:

  • normal - обычный вид строчных букв
  • small-caps - выглядят, как прописные буквы, уменьшенные до высоты строчных
.title {font-variant: small-caps;}
...
<div class="title">The Great Gatsby</div>

Выглядит в браузере, как:
The Great Gatsby

Свойство font-weight

Свойство font-variant толщину букв на экране:

  • normal, bold - браузер определяет, как выглядят нормальные или жирные буквы
  • bolder - текст, окруженный контейнером со свойством bolder выглядит более жирным, чем остальной текст
  • lighter - текст, окруженный контейнером со свойством lighter выглядит более тонким, чем остальной текст
  • 100, 200, 300, 400, 500, 600, 700, 800, 900 - 400 такой же, что и нормальный текст, а 700 такой же, что и жирный текст.

Свойство font-size

Свойство font-size задает размер шрифта текста:

  • xx-small, x-small, small, medium, large, x-large, xx-large - браузер определяет размер шрифтов xx-small, x-small, small и т.д.
  • smaller - более мелкий шрифт, чем окружающий
  • larger - более крупный шрифт, чем окружающий

Шрифты чаще всего определяют в долях от нормальной высоты строчных букв:
.disclaimer {font-size: .5em;}
.advertisement {font-size: 3em;}

Шрифты можно определять в абсолютных единицах:
in, cm, mm, px, pt (point) или pc (pica)
где in = 6pc

Шрифты можно определять в относительных единицах:
em, rem, ex
Один rem (от «root em», т.е. «корневой em» или «em корневого элемента») — это размер шрифта корневого элемента в документе. В отличие от em, который может быть для каждого элемента свой, rem для всего документа один и тот же.
Относительно новые единицы vw (view width) и vh (view height) определяют 1/100 часть (процент) от ширины и высоты видимого окна. Есть также единицы vmax и vmin.

Свойство font-family

.ascii-art {font-family: Courier, Prestige, monospace;}
Последний шрифт в списке должен быть обобщенным шрифтом, вариант которого всегда имеется в операционной системе. Название такого текста начинается со строчной буквы и оно указывает компьютеру выбрать имеющийся шрифт из указанного семейства.
Для того, чтобы шрифты отображались одинаково в разных браузерах, необходимо, чтобы все выбранные для данного свойства шрифты происходили из одного семейства шрифтов.
Браузеры распознают и поддерживают пять семейств обобщенных шрифтов:

Название семейства шрифтов Описание шрифта Пример шрифта
monospace У всех букв одинаковая ширина Шрифт Courier New
serif Буквы имеют украшения Шрифт Times New Roman
sans-serif У букв отсутствуют украшения Шрифт Arial
cursive Имитирует написанный от руки текст Шрифт Monotype Corsiva
fantasy Декоративный и игривый стиль шрифта Шрифт Impact

Определение значений для шрифта без указания свойств шрифта

Можно использовать сокращенную форму определения значений для шрифта:
  blockquote {font: italic large "Arial Black", Helvetica, sans-serif;}

{font: font-style font-variant font-weight font-size font-family;}

Для этого берется обобщенное свойство для шрифтов font и к нему прилагаются все значения нужных свойств разделенные пробелом без запятых. Можно указывать значения не для всех свойств, но для свойств font-size и font-family значения обязательны для указания. В вышеуказанном примере приведены обязательные значения large и "Arial Black", Helvetica, sans-serif

Cвойства текста

Свойство line-height

Свойство line-height относится не к самому шрифту, а к расстоянию между строками, то есть это свойство текста, а не шрифта. Его можно указывать в единицах em или в процентах. 3em это то же самое, что 300%
.proof-reading {line-height: 300%;}

Свойство text-align

Свойство text-align определяет горизонтальное выравнивание блока текста
Свойство имеет следующие значения:
Значения свойства text-align Описание свойства
left Выравнивает текст по левому краю
right Выравнивает текст по правому краю
center Выравнивает текст по центру
justify Вытягивает строки от левого до правого края, выравнивая текст по обеим сторонам

Свойство text-decoration

Свойство text-decoration определяет декоративные элементы текста
Свойство имеет следующие значения:
Значения свойства text-decoration Описание свойства
none Никаких декоративных элементов
underline Подчеркивание текста
overline Линия над текстом
line-through Перечеркивание текст
blink Мигающий текст

Свойство text-transform

Свойство text-transform определяет строчные или прописные буквы шрифта
Оно имеет следующие значения:
Значения свойства text-transform Описание свойства
none Выводится обычный текст
capitalize Первая буква каждого слова становится прописной
uppercase Все буквы становятся прописными
lowercase Все буквы становятся строчными

Свойство text-indent

Свойство text-indent определяет размер отступа первой строки блока текста:
p {text-indent: 4em;}
На данной веб-странице используется отступ 1em

Свойства границ

Свойство border-style

Свойство border-style определяет тип границы, которая окружает соответствующий элемент
Ниже указаны значения свойства border-style:

Значения свойства border-style Описание свойства
none Браузер не показывает никаких границ. Это значение по умолчанию.
solid
dashed
dotted
double

Свойство border-width

Свойство border-width определяет толщину границы, которая окружает соответствующий элемент
Ниже указаны значения свойства border-width:

Значения свойства border-width Описание свойства
thin, medium, thick Браузер определяет толщину границы. По умолчанию - medium.
Количество единиц px Единица пикселя в CSS является размером пикселя на мониторе, когда коэффициент зуммирования находится в положении по умолчанию 100%
Чтобы использовать свойство border-width необходимо включить одно из значений свойства border-style, чтобы граница стала видимой. 
  .boxed {
    border-style: solid;
    border-width: 4px 2px 0px 2px;
    border-color: rgb (128,128,255);
  }

Свойство border-color

Свойство border-color определяет цвет границы выбранного элемента. Цвет можно выбрать по цветовым системам RGB, RGBA, HSL, HSLA.

My idol is Tim Berners-Lee. He rocks!

Использование border для сокращенной установки значений свойств

Так же, как и для сокращенной записи свойств font значения свойств пишутся через пробелы, без запятых.

    .understated-box {border: thin dotted blue;}
    .in-your-face-box {border: 10px solid;}

Использование свойств padding и margin, а также закругления border-radius

    .hot {background-color: red;}
    .cold {background-color: blue;}
    .label {
    color: white;
    font: bold xx-large Lucida, monospace;
    border: solid black;
    border-radius: 50%;
    padding: 20px;
    margin: 20px;
    display: inline-block;
    }

    <span class="hot label">HOT</span>
    <span class="cold label">COLD</span>
HOT COLD

Использование процентного отношения для отступа веб-страницы

Для свойства margin иногда лучше использовать не абсолютные значения в пикселях (px), а относительные значения в процентах. Например, когда необходимо сделать отступы для всей страницы, которые будут меняться по мере изменения ширины окна браузера:
  body {margin: 10%;}
