Маркированный список (unordered list)

Следующий код демонстрирует все организационные элементы:

      aside {
        border: thin dashed red;
        color: red;
        background-color: white;
        float: right;
        width: 200px;
        margin: 5px;
        padding: 10px;
      }
Строчка float: right; помещает элемент <aside></aside> справа от остальных элементов контейнера <section></section>, а строчка width: 200px; уменьшает ширину данного элемента до 200 пикселей.

Элемент <nav></nav> использует элементы <a></a> для навигации по веб-странице

Элемент <footer></footer> часто содержит в себе такие данные, как авторское право, информация об авторе, связанные ссылки и т.д. В данном элементе часто располагают элемент <address></address>, который предоставляет контактную информацию для всей веб-страницы. Если элемент элемент <address></address> размещают внутри элемента <article></article>, то контактная информация дается только для статьи <article></article>. Сам по себе элемент <address></address> является блочным элементом и чтобы его поместить в ряд с другими элементами, необходимо изменить его поведение.
