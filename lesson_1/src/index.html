Основы языка гипертекстовой разметки HTML

Язык гипертекстовой разметки состоит из элементов разметки. Такие элементы позволяют создавать желаемый внешний вид страницы в браузере

Элементы кода веб-страницы

Элементы кода делятся на два основных типа - парные элементы (container elements), например <h3></h3>, <p></p>, <body></body> и непарные элементы (void elements), такие как <br>, <meta>, <hr>

Служебный непарный элемент <!DOCTYPE html>

<!DOCTYPE html> — Сообщает браузеру тип веб-страницы. Браузер понимает, что синтаксис страницы подчиняется стандарту HTML5.1

Парный структурный элемент <html lang="ru"></html>

Задает внешний вид и правила для всей веб-страницы. Аттрибут lang="ru" указывает на то, что веб-страница написана на русском языке

Внутри структурного элемента <html></html> располагаются два парных структурных элемента <head></head> и <body></body>

Парный структурный элемент <head></head>

Содержит служебную информацию о веб-странице в парных и непарных элементах:

  1. Непарный элемент <meta> обеспечивает браузер информацией о веб-странице (бинарная кодировка используемых символов UTF-8, имя автора, описание страницы, ключевые слова для поисковых машин).
  2. Парный элемент <title><title> обеспечивает браузер названием веб-страницы, указываемой в строке заголовка закладки браузера. Кроме этого, он (1) обеспечивает документацию для того, кто обслуживает веб-страницу и (2) помогает поисковым машинам типа Google найти данную веб-страницу
  3. Важным парным элементом является элемент <style></style> который обеспечивает стилистическое оформление веб-страницы. Обычно такой элемент содержит ссылку на внешний файл стиля в формате rel="stylesheet" href="локальный или сетевой адрес файла стиля/style.css", реже данный элемент содержит в себе сам код в формате языка стилей CSS.
  4. Другим важным элементом является элемент <script></script>, который обеспечивает код, управляющий элементами веб-страницы по правилам языка Java Script. Обычно такой элемент содержит в себе адрес, указывающий браузеру место хранения файла с управляющим кодом. Реже код пишется прямо в самом элементе <script></script> в нижней части элемента заголовка <head></head>.

Парный структурный элемент <body></body>

Парный элемент <body></body> содержит всю информацию веб-страницы, которую видно в окне браузера.
Тело страницы содержит в себе следующие типичные элементы:
<p></p>
<div></div>
<span></span>
